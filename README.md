# Discord WebHook Tool

## Features:
* Webhook info
    - Prints you the infos about the webhook
* Webhook spammer
    - Check if webhook is valid before continuing
    - Load from files ( MULTIPLE WEBHOOKS )
    - Multi-Threaded
* Webhook deleter
    - Deletes the webhook(s)
    - Load from files ( MULTIPLE WEBHOOKS )
    - Send a custom message before deleting
